# Hue

Open source SQL Query Assistant service for Databases/Warehouses.

**[Website](https://gethue.com//)** | **[Documentation](https://gethue.com/)** | **[GitHub](https://github.com/cloudera/hue)**

## How to enable?

```
platys init --enable-services HUE
platys gen
```
