# Presto

Distributed SQL Query Engine for Big Data.

**[Website](https://prestodb.io/)** | **[Documentation](https://prestodb.io/docs/current/)** | **[GitHub](https://github.com/prestodb/presto)**

## How to enable?

```
platys init --enable-services PRESTO
platys gen
```
