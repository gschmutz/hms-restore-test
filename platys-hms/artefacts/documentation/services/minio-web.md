# Minio Web

Web server for S3 compatible storage.

**[Documentation](https://github.com/e2fyi/minio-web)** | **[GitHub](https://github.com/e2fyi/minio-web)**

## How to enable?

```bash
platys init --enable-services MINIO, MINIO_WEB
platys gen
```
