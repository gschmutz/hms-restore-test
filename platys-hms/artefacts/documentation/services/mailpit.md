# Mailpit

An email and SMTP testing tool with API for developers.

**[Documentation](https://github.com/axllent/mailpit/wiki)** | **[GitHub](https://github.com/axllent/mailpit)**

## How to enable?

```
platys init --enable-services MAILPIT
platys gen
```
