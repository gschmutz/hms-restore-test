# Attu

Attu is an all-in-one Milvus administration tool. With Attu, you can dramatically reduce the cost of managing milvus.

 **[GitHub](https://github.com/zilliztech/attu)**

## How to enable?

```
platys init --enable-services MILVUS,ATTU
platys gen
```
