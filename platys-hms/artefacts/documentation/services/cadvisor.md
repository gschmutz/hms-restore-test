# cAdvisor

Analyzes resource usage and performance characteristics of running containers. 

**[Documentation](https://github.com/google/cadvisor)** | **[GitHub](https://github.com/google/cadvisor)**

## How to enable?

```
platys init --enable-services CADVISOR
platys gen
```
