# MailHog

Web and API based SMTP testing.

**[Documentation](https://github.com/blueimp/mailhog)** | **[GitHub](https://github.com/blueimp/mailhog)**

## How to enable?

```
platys init --enable-services MAILHOG
platys gen
```

Add MONGO if you want to use MongoDB for storage.
