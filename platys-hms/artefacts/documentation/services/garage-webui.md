# Garage WebUI

WebUI for Garage Object Storage Service.

**[Documentation](https://github.com/khairul169/garage-webui)** | **[Git](https://github.com/khairul169/garage-webui)**

## How to enable?

```
platys init --enable-services GARAGE GARAGE_WEBUI
platys gen
```
