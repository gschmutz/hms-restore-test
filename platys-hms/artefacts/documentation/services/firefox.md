# Firefox

Firefox running as a Docker container.

**[Website](https://www.mozilla.org/en-US/firefox/new/)** | **[Documentation](https://www.mozilla.org/en-US/firefox/new/)** 

## How to enable?

```
platys init --enable-services FIREFOX
platys gen
```
